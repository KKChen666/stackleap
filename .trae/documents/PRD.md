# 项目介绍网站 - 产品需求文档

## 1. 产品概述

一个视觉效果出色、交互体验流畅的项目介绍网站，通过动态背景、粒子动效等前沿设计手段展示个人/团队项目成果。

- 目标：打造具有科技感的项目展示平台，提升项目吸引力与专业度
- 目标用户：潜在合作伙伴、招聘方、技术社区访客
- 核心价值：通过沉浸式视觉体验展示技术实力与项目成果

## 2. 核心功能

### 2.1 功能模块

1. **首页**：Hero区域（动态粒子背景）、精选项目展示、关于我/团队简介
2. **项目列表页**：项目卡片网格、技术栈筛选、关键词搜索
3. **项目详情页**：项目完整信息、开发过程、技术亮点、动态背景场景
4. **关于我/团队页**：个人/团队介绍、技能展示、经历时间线
5. **联系我们页**：联系表单、社交链接

## 3. 核心流程

### 3.1 用户浏览流程

用户访问首页 → 浏览动态背景与精选项目 → 点击进入项目列表页 → 筛选/搜索感兴趣的项目 → 点击项目卡片进入详情页 → 查看完整项目信息 → 通过导航访问关于我/联系我们

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深空蓝（#0a0e27）为底，霓虹蓝（#00d4ff）与电紫（#7b2ff7）为主强调色
- **辅助色**：暗灰（#1a1f36）用于卡片背景，浅灰（#8892b0）用于次要文字
- **按钮风格**：圆角矩形（border-radius: 8px），渐变背景（蓝→紫），hover时光晕扩散效果
- **字体**：
  - 标题：使用 "Space Grotesk" 或类似科技感无衬线字体，font-weight: 700
  - 正文：使用 "Inter" 或 "Noto Sans SC"，font-weight: 400/500
  - 代码/技术栈标签：使用等宽字体
- **布局**：顶部固定导航栏（滚动时背景渐变），全宽Hero区域，卡片网格布局
- **图标**：使用 Lucide React 图标库，线条风格
- **特效**：粒子动态背景、卡片悬浮发光边框、文字打字机效果、滚动触发动画

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 首页 | Hero区域 | 全屏Three.js粒子背景，居中大标题（打字机动画），渐变叠加层，向下滚动箭头 |
| 首页 | 精选项目 | 3列卡片网格，悬浮时卡片上移+发光边框，图片区域渐变遮罩 |
| 首页 | 关于简介 | 左文右图布局，技能进度条动画 |
| 项目列表页 | 筛选栏 | 标签式筛选按钮组，搜索输入框带图标 |
| 项目列表页 | 项目网格 | 3列响应式卡片，加载时从下方渐入，卡片含封面图+技术栈标签 |
| 项目详情页 | 项目信息 | 全宽Banner图，叠加项目名称，技术栈标签横向排列 |
| 项目详情页 | 开发过程 | 垂直时间线，左右交替排列里程碑节点 |
| 关于我 | 个人介绍 | 头像圆形裁剪+发光边框，技能雷达图或环形进度图 |
| 联系我们 | 联系表单 | 暗色表单卡片，输入框底部边框线风格，提交按钮渐变 |

### 4.3 响应式设计

- **桌面端（≥1200px）**：3列项目卡片，完整导航栏
- **平板端（768px-1199px）**：2列项目卡片，导航栏保持
- **手机端（<768px）**：1列项目卡片，汉堡菜单导航，触摸优化的卡片间距
- 动态背景在移动端降低粒子数量以保证性能

### 4.4 3D 场景指引

- **粒子动态背景**：
  - 环境：深空主题，暗色背景上的发光粒子
  - 粒子系统：Three.js Points材质，数量约1000-2000个粒子
  - 粒子颜色：霓虹蓝（#00d4ff）与电紫（#7b2ff7）渐变
  - 相机：透视相机，缓慢旋转+鼠标跟踪视差
  - 交互：鼠标移动时粒子产生扰动效果，滚动时粒子密度变化
  - 后处理：Bloom发光效果，增加科技感
  - 性能：移动端粒子数降至500-800，使用requestAnimationFrame控制帧率
