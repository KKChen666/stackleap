# StackLeap

StackLeap is a Vite, React, and TypeScript single-page site for a full-stack technology team.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- GSAP for scroll and entrance motion

## Features

- Chinese and English content switcher
- Project cards with detail dialogs
- Capability matrix with delivery context
- Contact form with client-side validation
- Responsive navigation for desktop and mobile

## Scripts

```bash
npm run dev
npm run check
npm run lint
npm run build
npm run preview
```

## Development

Install dependencies, then start the local server:

```bash
npm install
npm run dev
```

## Production

Create a production build:

```bash
npm run build
```

Preview the production output locally:

```bash
npm run preview
```
